🔍 How to Test the Promotion Modal

Step 1: Open the admin portal at http://localhost:5177
Step 2: Navigate to the Services page from the sidebar
Step 3: Find any service in the table
Step 4: Click the purple Package icon (📦) in the Actions column
Step 5: The promotion settings modal will appear!

🎯 Available Promotion Categories

1. Top Search Results (0-100)
Higher values make the service appear first in search results
2. Category Priority (0-100)
Higher values make the service appear first in category listings
3. Enhanced Listing
Toggle to show larger images and more details
4. Increased Visibility
Toggle to make the service appear in more sections across the platform
5. Top Carousel Placement (0-100)
Higher values make the service appear first in the homepage carousel
6. Maximum Visibility
Toggle to give the service maximum visibility everywhere
7. Promotion Expires At
Optional date when promotion settings will expire

💡 Tips

  • If you don't see the modal, try refreshing the page (Ctrl+F5 or Cmd+Shift+R)
  • Make sure the backend server is running on port 5000
  • The purple Package icon (📦) is in the Actions column of each service row
  • You can test with any service - the modal will show current promotion settings
