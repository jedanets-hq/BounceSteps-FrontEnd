# 🚀 Quick Start - BounceSteps Deployment

## Kila Unapofanya Changes

```bash
./deploy.sh
```

Hii script itafanya:
1. ✅ Pull latest changes
2. ✅ Add na commit changes zako
3. ✅ Push kwa GitHub
4. ✅ Trigger Vercel deployment

---

## Kama Deploy Script Inashindwa

```bash
# Refresh authentication
gh auth refresh -s repo,workflow

# Jaribu tena
./deploy.sh
```

---
